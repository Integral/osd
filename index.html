Словарь

Для активации режима реверсивной сортирвоки нажмите на кнопку "Reverse Sorting", после этого данные будут сортироваться в обратном порядке. Для того чтобы вернуться к обычной сортировке нажмите снова на кнопку "Reverse Sorting" или кнопку "Reset".

Для сохранения полученной выборки вспользуйтесь кнопкой "Download". Исходные данные словаря с индексами для обычной и реверсивной сортировок доступны в формате CSV и XLS, для воспроизведения алфавита Вам понадобится установить шрифт.
